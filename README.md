## Project Overview

This project presents an end-to-end **Customer Shopping Behavior Analysis** solution designed to help a retail company better understand consumer purchasing patterns and make data-driven business decisions. Using a dataset of 3,900 customer transactions, the project combines **Python**, **PostgreSQL (SQL)**, and **Power BI** to transform raw transactional data into actionable business insights. The workflow includes data cleaning, preprocessing, feature engineering, exploratory data analysis, and database integration using Python, followed by SQL-based business analysis to identify customer segments, purchasing trends, revenue drivers, and product performance. Finally, an interactive Power BI dashboard visualizes key performance indicators, customer demographics, sales distribution, subscription behavior, discount effectiveness, and product preferences, enabling stakeholders to make informed marketing, customer engagement, and inventory management decisions. The repository contains the complete source code, SQL scripts, dashboard files, project report, and supporting documentation, providing a comprehensive example of an end-to-end retail analytics project that demonstrates practical applications of data analytics in solving real-world business problems.
